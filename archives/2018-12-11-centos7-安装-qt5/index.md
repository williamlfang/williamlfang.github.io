# CentOS7 安装 Qt5


## 安装连接器

```bash
yum -y install mesa-libGL-devel mesa-libGLU-devel freeglut-devel
```

# 下载 `Qt` 软件包

从网站[下载](http://download.qt.io/archive/qt/)得到需要的软件包，我这里选择 `Qt5.10.1`
