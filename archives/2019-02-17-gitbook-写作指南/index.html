介绍如何使用 gitbook 编写技术手册，并配置 github 实现对内容的版本控制。

gitbook 是在线文本的展示与管理网站，提供了独立主页、项目结构、文本形式等规范化的项目格式，从而使得我们只需要专注于内容的生产，无需处理格式化的问题，从而实现了快速发布文档。
