# CppServer


使用 CppServer 实现 TCP/UDP 通信。


# 安装

```bash
apt-get install uuid-dev

pip3 install gil

git clone https://github.com/chronoxor/CppServer.git
cd CppServer
gil update

cd modules
rm -rf asio
pc git clone git@github.com:chriskohlhoff/asio.git
```

&lt;!--more--&gt;


# 编译

```bash
cd build
./unix.sh
```
